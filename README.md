# Soulsborne-Inspired Open World 3D Game

A seamless open-world 3D action RPG inspired by Elden Ring and the Soulsborne series, built with Three.js.

## Features (Planned)
- Dodge rolling with i-frames
- Stamina-based combat system
- Seamless open world exploration
- Challenging enemies and boss encounters
- Character progression and leveling
- Atmospheric visuals and lighting

## Controls
- WASD: Movement
- Mouse: Camera
- Space: Dodge Roll
- Left Click: Attack
- Right Click: Block
- Shift: Sprint
- Tab: Lock-on

## Development
This game is being developed using Three.js for browser-based 3D rendering.
